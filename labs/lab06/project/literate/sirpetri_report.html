Цель

Построить итоговые графики для отчёта: сравнение детерминированной и стохастической динамики, а также чувствительность пика инфицированных к β.

Загрузка проекта

using DrWatson
@quickactivate "project"

using DataFrames, CSV, Plots

include(srcdir("SIRPetri.jl"))
using .SIRPetri

Загрузка результатов

df_det = CSV.read(datadir("sir_det.csv"), DataFrame)
df_stoch = CSV.read(datadir("sir_stoch.csv"), DataFrame)
df_scan = CSV.read(datadir("sir_scan.csv"), DataFrame)

Сравнение траекторий

p1 = plot_infected_comparison(df_det, df_stoch)
savefig(p1, plotsdir("comparison.png"))
p1

Чувствительность к β

p2 = plot(
    df_scan.β,
    df_scan.peak_I,
    marker = :circle,
    xlabel = "β",
    ylabel = "Peak I",
    title = "Sensitivity of epidemic peak to β",
    linewidth = 2,
    label = "Peak I",
)
savefig(p2, plotsdir("sensitivity.png"))
p2

Вывод

Сравнительный график показывает близость детерминированной и стохастической динамики при большой популяции. График чувствительности подтверждает, что снижение β уменьшает пик эпидемии и является ключевым механизмом управления распространением инфекции.
